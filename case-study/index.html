OVERVIEW


Although very straight to the point when finally clicked on by the user, the original dog slideshow on Petfinder lacked warmth, clarity, and intuitive interaction, making it difficult to fully understand both the breed and the intended micro interaction experience. With no clear navigation cues, minimal motion feedback, and inconsistent animation states among slides, users had little guidance or motivation to explore.

During the redesign period, the site updated its slideshow; but the new layout became visually fragmented with elements misaligned, irregular spacing, and content occasionally cut off, further emphasizing the need for a more structured interaction model.

Over a ten week time frame, I redesigned an HTML-based prototype that focused on creating a friendlier, more dog-centered interaction. By introducing softer transitions, paw-shaped navigation controls, restructuring the slideshow layout, and adding playful animations across the five interactive models, the updated experience improves affordance and creates a warmer, more emotional tone that better aligns with the content being presented.

These changes transform the slideshow from a static image viewer into a more exploratory and engaging moment within the site, encouraging users to meaningfully interact with the breed visuals rather than simply scrolling past them.

    • RULES

    Object 1 (MAIN LARGE IMAGE) Hover On
  • The main image increases in brightness by 15% over 0.35 seconds.
  • The image scales up by 7% within 0.35 seconds.
  • The cursor changes to a pointer.

    • Hover Off
  • The main image returns to its original brightness and size within 0.35 seconds.
  • The pointer changes back to a cursor.

    • Click
  • When the main image is clicked, the same image source fades in as the light box image over 0.35 seconds.
  • The main image scales up to 101.5% while rising 10px and returns to its original state in 0.65 seconds.
  • The paw arrows, close button, and navigation dots fade in within 0.35 seconds.

    • Object 2 (LIGHTBOX ARROWS) Hover On
  • The selected paw arrow increases to a brightness of 200% in 0.35 seconds.
  • The cursor changes to a pointer.

    • Hover Off
  • The paw arrow reverts back to its original brightness in 0.35 seconds.
  • The pointer changes back to a cursor.

    • Click
  • The next lightbox image replaces the previous one in 0.25 seconds. The previous image fades out during the transition in the 0.25 seconds and the new one fades in at 0.25 seconds.
  • The selected paw arrow will rotate 10 degrees in one direction and then swing back in 0.65 seconds.
  • The paw arrow shrinks to 80% of its size and returns to 100% in 0.6 seconds.
  • The navigation dots fade into a purple hue within 0.5 seconds, updating to indicate the currently active image.
    • Object 3 (CLOSE BUTTON) Hover On
  • The close button decreases in opacity by 35% in 0.4 seconds.
  • The cursor changes to a pointer.

    • Hover Off
  • The close button reverts back to its original opacity in 0.4 seconds.
  • The pointer changes back to a cursor.

    • Click
  • The close button moves vertically by 10px and rotates 15° within 0.35 seconds.
  • During this transition, the lightbox background, x button, dots, and select image fade out in 0.5 seconds.
  • Everything then reverts to the original gallery view, with the last selected lightbox image displayed as the main image.
  • The main new selected image will fade in to replace a previous one in 0.35 seconds.
