RULES

    Object 1
      Hover On
      • Cursor changes from arrow to finger.
      Hover Off
      • Cursor changes back to an arrow.
      Click
      • When the main image is clicked, it becomes the active element and opens in a lightbox view.
      • The active image enlarges and centers on the screen for focused viewing.
      • A dark overlay appears behind the image
      • Navigation arrows and a close (“X”) icon appear for browsing and exiting the lightbox.

    Object 2
      Hover On
      • Cursor changes from arrow to finger.
      Hover Off
      • Cursor changes back to an arrow.
      Click
      • Clicking the right arrow advances to the next image in the sequence.
      • Clicking the left arrow returns to the previous image.
      • The small indicator dots below the image update to show which photo is currently active.
      • Only one image is displayed at a time. Each new image replaces the previous one smoothly.

    Object 3
      Hover On
      • Cursor changes from arrow to finger.
      Hover Off
      • Cursor changes back to an arrow.
      Click
      • The lightbox fades out, and the background lightens back to white.
      • The user sees the original page layout return to view instantly.
      • All additional lightbox UI elements (arrows, dots, and “X”) disappear at once for a clean exit.

FEEDBACK

    Object 1
      Hover On
      • The cursor visually changes to a hand icon, signaling that the main image is clickable.
      Click
      • The main image smoothly enlarges into a centered view in 0.5 seconds.
      • A dark overlay appears behind the image with 80% #000000 opacity, dimming the rest of the interface to direct the user’s attention solely to the active content.
      • Navigation arrows, dots, and the close (“X”) icon fade in around the image in 0.15 seconds, signaling that the user has entered a new interactive mode (the lightbox) with new available actions.

    Object 2
      Hover On
      • When hovered, the cursor changes to a hand icon over the navigation arrow paws, indicating that they are clickable.
      • The navigation arrow paw rotates by 15 degrees when hovered on.
      Click
      • The image slides left or right to show the next or previous photo in 0.5 seconds. The next image in the transition slides to replace the previous image in 0.5seconds alongside it.
      • The active pagination dot updates to match the new image.
      • When clicking the navigation arrow paw, it moves up and down by 10 degrees (y-axis) in 0.15 seconds.

    Object 3
      Hover On
      • When hovered, a rectangular outline appears around the “X” icon, and the cursor changes to a hand icon, indicating it is clickable.
      Click
      • The lightbox fades out quickly and the background returns to its original color.
      • All additional UI elements disappear, returning the user to the normal browsing state.
