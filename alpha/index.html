URL TO USER INTERFACE

https://www.petfinder.com/dogs-and-puppies/breeds/canadian-eskimo/

TRIGGERS

User is browsing multiple breeds and comes to one they are particularly interested in. The user would like to browse this breed's photos to reinforce their visual understanding.

RULES

  • When a thumbnail is activated, the main image swaps to the corresponding photo.
  • The selected thumbnail is marked as active (purple outline) and previously active thumbnail is cleared.
  • The down chevron scrolls/expands the thumbnail list if more images are hidden below.

FEEDBACK

  • SEE - Main image is provided with a scroll up animation after the sub images are clicked on
  • HEAR - Nothing in relation to the section provides sound. User may hear the click of their mouse when looking through the images
  • FEEL - User will feel the click of their mouse or mousepad

LOOPS & MODES (meta-rules)

  • LOOPS - Users can repeatedly select thumbnails in any order; state always reflects the most recent selection.
  • BOUNDS - Image set is finite to the pageâ€™s gallery; the loop ends when the user navigates away.
  • MODE - Browsing mode (default) where the main image is view-only; no edit or zoom controls are present here.
  • PROGRESSIVE DISCLOSURE - The down arrow acts as a control to access the rest of the gallery when present.
  • ERROR PREVENTION - Only one thumbnail can be active at a time; invalid states are avoided by design.
